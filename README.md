# Presidential — Maison Digitale

A redesigned digital home for Presidential RX — the original infused-cannabis Maison, California, est. 2012.

This is a static, multi-page site — no build step. Open `index.html` in a browser (or serve the folder locally) to view.

## Structure

```
index.html          Entry · age gate · homepage hero
collection.html     The three signatures — Moon Rocks · Blunts · Prerolls
atelier.html        Presidential × THC Design — Atelier Edition
heritage.html       Brand story · terms · shipping & returns
dispensaries.html   Find us · Weedmaps locator
journal.html        Editorial · FAQ
contact.html        Concierge
css/presidential.css
js/presidential.js
assets/img/         Brand imagery (logo, product photography, heritage)
scrape/             Raw archive of the previous site (HTML, images, sitemap)
```

## Design system

- **Typography** — Bodoni Moda (display), EB Garamond (editorial), Inter (UI).
- **Palette** — Obsidian, Ivory, the Maison's authentic teal, champagne gold accent.
- **Motion** — Restrained: fade-in scroll reveals, slow marquee, animated scroll cue.
- **Architecture** — Full-bleed cinematic heroes, editorial 2-up blocks, 3- and 4-column grids, generous whitespace, hairline rules, small-caps eyebrows.
